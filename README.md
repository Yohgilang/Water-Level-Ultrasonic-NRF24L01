# Water Level Controller with Data Telemetry
This is a telemetry project to measure water levels using ultrasonic sensors and NRF24L01

Components:
- Module NRF24L01
- Arduino Uno
- Ultrasonic HC-SR04
- LCD Crystal 16 x 2
- I2C LCD
